# Tic tac toe


## Description

This is basic tic tac toe game written in c language. 

## How to use

Clone repository.
```bash
git clone https://github.com/zuhos/tic-tac-toe
```
Move to cloned repository
```
cd tic-tac-toe
```
Compile
```
gcc tic_tac_toe.c -o main
```

Run:

Linux
```
./main
```
Windows
```
main.exe
```
